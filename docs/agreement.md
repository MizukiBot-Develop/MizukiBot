# Mizuki Bot 用户使用协议

## 前言

欢迎使用 Mizuki Bot。本机器人由 **Mizuki Bot 开发组**（以下简称"开发组"）研发并运营，旨在为用户提供多元化的群聊交互功能，包括但不限于 Project SEKAI（PJSK）辅助、舞萌 DX（Maimai DX）成绩同步与查询、Bilibili 弹幕网 链接解析、网易云音乐点歌、图片与表情包生成、群聊经济系统及其他实用工具。
**Mizuki Bot 开发组**是由志愿者组成的非盈利技术团队，未进行商业注册。
如果您对本协议或本服务有意见或建议，可通过电子邮箱（help@mizuki.top）联系我们。
为使用 Mizuki Bot（以下简称"本机器人"）及服务（以下统称"本服务"），您应当阅读并遵守《Mizuki Bot 用户使用协议》（以下简称"本协议"）。您开始使用本机器人的任何功能，即视为您已阅读、理解并同意接受本协议所有条款的约束。如果您不同意本协议的任何内容，请立即停止使用本机器人。

---

## 1. 定义

**1.1 用户**：指通过 腾讯QQ 与本机器人进行交互的自然人。
**1.2 开发组**：指负责 Mizuki Bot 的研发、维护及运营的团队。
**1.3 服务**：指 Mizuki Bot 通过 腾讯QQ 平台提供的各项功能，包括但不限于本协议序言所述内容。
**1.4 第三方服务**：指本机器人接入的、由第三方提供的 API 或服务，例如水鱼查分器、落雪咖啡屋、网易云音乐等。

---

## 2. 服务使用规范

**2.1 账号绑定**：用户在使用本机器人的核心功能前，必须使用 "qbind" 指令将其 QQ 账号与本机器人进行绑定。未完成绑定的用户将无法使用相关核心功能。
**2.2 使用范围限制**：为保障机器人主进程的稳定运行及账号安全，本机器人已禁用所有功能的私聊响应。用户应当在 QQ 群聊环境中使用本机器人。频繁或恶意对机器人的私聊行为均可能导致用户账号被列入监控名单并遭受封禁。
**2.3 功能合理使用**：用户在使用"聊天截图伪造"、"喜（悲）报生成器"等可自定义生成内容的功能时，应当遵守国家法律法规及社会公序良俗。用户不得利用本机器人制作、传播任何违法、违规或可能引发群体性事件的信息。用户应对其生成及传播的内容承担全部责任。

## 3. 禁止行为与黑名单机制

为确保维护服务的稳定性及开发组的合法权益，开发组设立严格的黑名单机制。若用户发生以下任一行为，开发组有权在不事先通知的情况下，立即将该用户的 QQ 账号列入黑名单，并视情况决定是否将其所在群聊列入黑名单，停止提供任何服务。
**3.1 滥用服务**：使用任何自动化脚本、外挂程序、连点器或采用其他人为手段，以超出正常频率的方式发送指令，导致本机器人服务异常、响应迟缓或服务器资源被过度消耗。
**3.2 传播违规内容**：利用本机器人制作、存储、传播任何包含色情、暴力、恐怖、赌博、煽动性政治内容或其他违反中华人民共和国法律法规的信息。
**3.3 攻击与骚扰**：利用系统漏洞对本机器人后端进行攻击、渗透或破坏；对开发组成员或其他用户进行辱骂、威胁、诽谤、网络暴力或人身攻击。
**3.4 其他严重影响**：经开发组合理判断，用户的行为已严重干扰本机器人的正常运行，或对开发组成员的日常生活、身心健康造成实质影响。
**3.5 申诉机制**：用户如对黑名单处理有异议，可通过邮箱或表单向开发组提出申诉。对于“恶意请求”和“违规行为”的认定，开发组将基于客观事实和服务器日志进行合理判断。

## 4. 隐私与数据保护

**4.1 数据收集**：我们仅收集提供服务所必需的最少数据，包括 QQ 账号、绑定的第三方账号信息及指令使用记录。
**4.2 数据用途**：所收集数据仅用于提供本服务功能，不会用于商业目的或向第三方出售。

**4.3 数据共享**：为提供本服务，我们可能会将您的必要信息（如指令内容、绑定账号）传输至合作的第三方服务商，但仅限于实现服务功能所需的范围，并要求其遵守数据保护义务。

**4.4 数据存储**：数据存储于中国大陆境内服务器，保存期限不超过服务所需时间。

**4.5 用户权利**：用户可通过邮箱联系我们，请求查询、更正或删除个人数据。
**4.6 数据安全**：我们采取合理的技术措施保护用户数据安全，但无法保证绝对安全。

**5.1** 若用户未满 14 周岁，应在监护人指导下使用本服务。
**5.2** 监护人不同意本协议内容的，未成年人应立即停止使用。
**5.3** 我们不会故意收集 14 周岁以下未成年人的个人信息，如发现误收集，将尽快删除相关数据。
