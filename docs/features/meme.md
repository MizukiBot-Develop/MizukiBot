# 🎵 表情包系统 (Meme)

> 集成审核、溯源、压缩包批量上传的表情管理系统。

<br>

## 🛠️ 基础指令

* `表情帮助` / `表情列表`
* `发送 [文件夹名]` (例: `发送 mzk`)
* `查看 [图片名]` (例: `查看 mzk1`)
* `看所有 [文件夹名]` (生成预览图)

<br>

## 📤 上传系统

* `[文件夹] 上传 [图片]` (Bot本地库)
* `[文件夹] pic上传 [图]` (Mzk API库)
* *注：Kuma API 已移除*

<br>

## 🛡️ 管理员指令

> 需要管理员权限

* `最近上传` - 查看入库记录
* `删除 [图片名]` - 删图+删库
* `溯源 [图片名]` - 查上传者
* `锁定/解锁 [文件夹]`
* `屏蔽群/解除屏蔽 [群号]`

<br>

## 🎨 PJSK 表情制作

> 使用表情 ID 快速生成 PJSK 风格对话贴纸。

**生成指令：**
`pjsk [ID] [文本]`

* 例: `pjsk 40 哇古哇古` (Emu)
* 例: `pjsk 280 呵呵` (Rui)
