# 舞萌 DX 专项

> 查分、查歌与排队服务。

<br>

## 🟡 核心查询

| 指令 | 说明 |
| :--- | :--- |
| `今日舞萌` | 运势与推荐 |
| `b50` | 查B50成绩 |
| `随个 [难度]` | 随机歌 (例: `随个紫`) |
| `查歌 [关键词]` | 查曲目信息 |
| `定数查歌 [范围]` | 例: `定数查歌 13.5 13.9` |

<br>

## 🟢 牌子与进度

| 指令 | 说明 |
| :--- | :--- |
| `[牌子]进度` | 例: `霸者进度` |
| `[等级]进度` | 例: `13+进度` |
| `完成表 [等级]` | 生成完成度表格 |
| `分数线 [曲ID]` | 查分数线详情 |

<br>

## 🔵 别名与娱乐

| 指令 | 说明 |
| :--- | :--- |
| `是什么歌` | 查别名对应曲 |
| `添加别名 [ID] [名]` | 添加新别名 |
| `mai猜歌` | 开启猜歌游戏 |
