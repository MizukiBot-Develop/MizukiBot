# 表情包系统 (Meme System)

<div class="custom-card" style="background: var(--vp-c-bg-soft); border: 2px solid #b37feb; padding: 20px; border-radius: 15px; margin-bottom: 25px;">
  <h3 style="margin-top: 0; color: #722ed1;">表情管理系统</h3>
  <p style="font-size: 13px; opacity: 0.8; margin-bottom: 15px;">集成审核、溯源、压缩包批量上传的表情管理系统。</p>
  
  <h4 style="color: #722ed1; margin-top: 15px;">基础指令</h4>

| 指令 | 说明 | 示例 |
| :--- | :--- | :--- |
| `表情帮助` | 查看系统帮助 | - |
| `发送 [文件夹]` | 随机发送一张 | `发送 mzk` |
| `查看 [图片名]` | 查看指定图片 | `查看 mzk1` |
| `看所有 [文件夹]` | 生成九宫格预览 | `看所有 mzk` |

  <h4 style="color: #722ed1; margin-top: 25px;">上传与管理</h4>

| 指令 | 说明 |
| :--- | :--- |
| `[文件夹] 上传 [图]` | 上传图片到本地库 |
| `[文件夹] pic上传 [图]` | 上传到 API 图床 |
| `最近上传` | (管理) 查看最新入库记录 |
| `删除 [图片名]` | (管理) 删除图片及记录 |
| `溯源 [图片名]` | (管理) 查看上传者QQ |
| `锁定/解锁 [文件夹]` | (管理) 控制文件夹权限 |

</div>
